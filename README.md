# CrystalDefence - Пуленепробиваемое стекло

Сайт компании по продаже пуленепробиваемых стекол для гражданского использования (частные дома, бизнес-центры, VIP-автомобили).

## 📁 Структура проекта

```
CrystalDefence_Site/
├── src/                    # Исходные шаблоны и ассеты
│   ├── templates/          # Jinja2 шаблоны для сборки
│   │   ├── base.html       # Базовый шаблон (навигация, футер)
│   │   ├── index.html      # Главная страница
│   │   ├── products.html   # Каталог продукции
│   │   └── about.html      # О компании
│   └── static/             # Статические файлы шаблонов
│       └── style.css       # CSS стили
├── docs/                   # Генерируемые HTML-страницы
│   ├── index.html         # Главная страница (сгенерированная)
│   ├── products.html      # Каталог (сгенерированный)
│   ├── about.html         # О компании (сгенерированный)
│   └── static/            # Статические файлы (CSS, etc.)
├── build.py                # Скрипт сборки сайта
├── requirements.txt        # Python зависимости
├── .nojekyll              # Маркер для GitHub Pages (не обрабатывать)
├── README.md              # Документация
└── .gitignore             # Игнорирование системных файлов

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Сборка сайта

```bash
python build.py
```

После выполнения этой команды в папке `docs/` будут сгенерированы:
- `index.html` — Главная страница
- `products.html` — Каталог продукции  
- `about.html` — О компании
- `static/style.css` — CSS стили

### 3. Публикация на GitHub Pages

1. Загрузите все файлы репозитория (включая папку `docs/`) в GitHub
2. В настройках GitHub Pages укажите ветку, куда загрузили файлы
3. Сайт будет доступен по адресу: `https://username.github.io/repository-name/`

## 🛠️ Технологии

- HTML5 + CSS3
- Jinja2 шаблоны для сборки

## 📱 Адаптивность

Сайт автоматически адаптируется под различные размеры экрана:
- 💻 Desktop (1200px+) — полная версия с навигацией
- 📱 Tablet (768px - 1200px) — адаптированная верстка
- 📱 Mobile (< 768px) — компактный интерфейс, скрытое меню

## 🔐 Безопасность

Все тексты на сайте ориентированы на гражданских лиц:
- ✅ Безопасность частных домов и семей
- ✅ Защита бизнес-центров и офисов
- ✅ Пуленепробиваемые стекла для VIP-автомобилей
- ⛔ Нет военных/военно-полицейских формулировок

## 📝 Обновление сайта

Чтобы обновить контент после внесения изменений в шаблоны:

```bash
# 1. Внесите изменения в src/templates/*.html и src/static/style.css
# 2. Пересоберите сайт
python build.py

# 3. Загрузите содержимое docs/ в GitHub (или деплойте на хостинг)
```
